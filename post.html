¿Qué es HTML?

HyperText Markup Language

Posted by OtroLenguaje el 17 de Marzo de 2022.

HTML corresponde a las siglas en ingles de HyperText Markup Language lo que en español se traduce como lenguaje de marcado de hipertexto. HTML es un estándar definido por el Consorcio World Wide Web (World Wide Web Consortium, abreviado como W3C) el cual proporciona una serie de elementos también llamados elementos HTML que se utilizan para desarrollar y estructurar una página web.

Conocer la historia de HTML nos ayudara a entender que es y para que se utiliza. En este orden de ideas, de manera breve daremos un repaso a los orígenes y evolución de este código:

El nacimiento de HTML

El código HTML surgió debido a las necesidades de información. El británico Tim Berners-Lee (Londres, Reino Unido, 8 de junio de 1955) es un científico de la computación considerado como el padre de la World Wide Web, quien en la década de los ochenta trabajaba en el Centro Europeo de Investigaciones Nucleares CERN y con el objetivo de distribuir e intercambiar información fácilmente de sus investigaciones entre científicos de distintas universidades, de centros de investigación y de organismo gubernamentales de todo el mundo, dio a conocer en 1990 el leguaje HTML como un subconjunto del SGML.

En 1986 SGML (en español lenguaje de marcado generalizado estándar y derivado de una GML de IBM)) se estandarizo por medio de la norma ISO 8879 y se convirtió en un metalenguaje, es decir que, describe formalmente un lenguaje de marcado fijando unas normas de vocabulario y gramática o la sintaxis que permitirían crear otro lenguaje. Entendamos que un lenguaje de marcado es una forma de codificar un documento o Web y definir la estructura del texto o su presentación para que pueda ser entendida por los navegadores se hace a través de etiquetas o marcas.

Continuando con los orígenes de HTML Tim Berners-Lee en 1990 también creó el proyecto World Wide Web que fue el primer navegador web ya que él quería crear una red que permitiera conectar e intercambiar informacion, además el también desarrollo el protocolo HTTP.

El concepto de “hipertexto” ya había sido introducido años atrás y este era un sistema que permitía a los usuarios acceder a la información relacionada con los documentos electrónicos que visualizan. Es decir, enlazaba o asociaba documentos con otros documentos, este sistema es fundamental en el uso de Internet.

El 1 de octubre de 1994 se fundó World Wide Web Consortium, organismo internacional que genera recomendaciones y estándares que aseguran el crecimiento de la World Wide Web a largo plazo y que publica las actualizaciones del lenguaje HTML desde 1996, ya que en 1995, Internet Engineering Task Force (IETF) organizó un grupo de trabajo de HTML y el 22 de septiembre publicaron el primer estándar oficial de HTML (HTML 2.0).

La versión HTML 3.2 se publicó el 14 de enero de 1997 y es la primera recomendación de HTML publicada por el W3C. El 24 de abril de 1998 (versión corregida de la publicación del 18 de diciembre de 1997) se publicó HTML 4.0 y lo más destacable de esta versión es que definieron cambios relacionados con las hojas de estilo, los formularios, scripts, imágenes, objetos y mapas de imágenes, entre otros e introdujeron nuevos elementos HTML.

La última especificación oficial de HTML se publicó el 24 de diciembre de 1999 y se denomina HTML 4.01, esta no incluye novedades significativas y el W3C se centró en el desarrollo del estándar XHTML lo que detuvo la evolución del HTML, en consecuencia, Apple, Mozilla y Opera anunciaron su interés en trabajar por el desarrollo de HTML y decidieron organizarse en una nueva asociación llamada WHATWG (Web Hypertext Application Technology Working Group) fundada en 2004.

Debido a la fuerza de las empresas que forman el grupo WHATWG y a la publicación de los borradores de HTML 5.0, en marzo de 2007 el W3C decidió retomar la actividad de estandarización de HTML y en la actualidad HTML5 ya es un estándar, el consorcio W3C con Tim Berners-Lee, el 27 de octubre de 2014 mostraron la versión final.

Por último, es importante resaltar que HTML es un lenguaje que no se utiliza solo, este debe ir acompañado de CSS Y JavaScript para darle estilo y dinamismo a una página web o documento HTML.

T¿Qué es un elemento HTML?

HTML define el término elemento para referirse a las partes que componen los documentos HTML. Es decir, el documento HTML es un todo y sus partes vendrían siendo los elementos que lo conforman, la mayoría de estos elementos se componen así:

1. Una etiqueta de apertura: estas comienzan con < seguido del nombre del elemento y finalizan con un>

2. Atributos: son palabras especiales utilizadas dentro de la etiqueta de apertura, para controlar o modificar el comportamiento del elemento.

3. Texto encerrado por la etiqueta: por lo general es el contenido que va a ser visible en la pagina web.
